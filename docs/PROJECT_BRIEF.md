# VANT Project Brief

## Purpose

Build and document an unmanned aerial vehicle with an emphasis on mechanical design, fabrication, test evidence, and iteration history.

## Current Phase

Concept and documentation setup.

## Primary Records

- `PROJECT_LOG.md`: chronological progress.
- `docs/decisions/`: design decisions.
- `docs/testing/`: test plans and results.
- `docs/fabrication/`: build and assembly notes.
- `BOM.md`: parts, materials, and sourcing.
- `wiki/`: stable project knowledge for GitHub Wiki.

## Open Questions

- Target aircraft configuration.
- Size, mass, and payload assumptions.
- Material and manufacturing process.
- Structural test approach.
- Safety boundaries for ground and flight testing.
